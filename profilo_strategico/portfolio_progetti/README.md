# Portfolio progetti strategici

Questo portfolio raccoglie i progetti più spendibili per:
- candidature EQ;
- concorsi Area Funzionari;
- posizioni in comunicazione, digital governance, terza missione;
- preparazione a profili dirigenziali.

## Progetti prioritari

1. Portale di Ateneo / CINECA
2. Redazione Web e Comunicazione Multimediale
3. AlmaLaurea / Alumni
4. USCEM / PNRR
5. Infiniti Mondi / Terza Missione e Innovazione

## Regola di aggiornamento

Ogni scheda deve evolvere da descrizione attività a prova di competenza manageriale.

Formula:
problema → ruolo → stakeholder → azione → risultato → impatto → evidenza.
