# Profilo target

## Posizionamento realistico
Manager universitario specializzato in comunicazione istituzionale, relazioni esterne, public engagement, terza missione e trasformazione digitale.

## Target prioritari
1. EQ / Area Funzionari comunicazione, relazioni esterne, terza missione.
2. Responsabile area comunicazione / public engagement / servizi digitali.
3. Dirigente sistemi informativi, comunicazione digitale, innovazione, governance.
4. Dirigente amministrativo universitario con forte componente digitale/relazionale.

## Punti forti
- comunicazione istituzionale;
- web e contenuti digitali;
- social e multimedialità;
- coordinamento ufficio;
- esperienza universitaria;
- progetti PNRR / portale / AlmaLaurea / public engagement;
- competenze AI e workflow.

## Gap da chiudere
- diritto amministrativo avanzato;
- pubblico impiego;
- contabilità pubblica;
- performance e valore pubblico;
- responsabilità dirigenziale;
- ordinamento universitario;
- casi pratici di gestione budget/personale/procedimenti.
