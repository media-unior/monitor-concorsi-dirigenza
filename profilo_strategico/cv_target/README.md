# CV target

Questa cartella contiene versioni orientate del profilo professionale.

## Versioni

1. `cv_eq_comunicazione.md`
   - Per bandi comunicazione, media, relazioni esterne, redazione web, social, multimediale.

2. `cv_eq_digital_governance.md`
   - Per bandi digitalizzazione, portali, dati, accessibilità, servizi digitali, innovazione.

3. `cv_terza_missione_public_engagement.md`
   - Per bandi terza missione, public engagement, stakeholder, territorio, orientamento, alumni.

## Regola

Non usare un CV unico per tutti i bandi.

Per ogni bando:
- leggere titolo e requisiti;
- scegliere CV target più vicino;
- adattare formula profilo;
- evidenziare solo progetti coerenti;
- non sovraccaricare con esperienze non pertinenti.
